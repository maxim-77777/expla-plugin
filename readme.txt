=== Custom Post Import ===
Автор: PM  
Версія: 1.0  
Опис: Плагін для автоматичного імпорту постів з API та виведення їх через шорткод.

 Опис 
Плагін щодня імпортує пости з зовнішнього API та зберігає їх у WordPress. Ви можете виводити ці пости за допомогою шорткоду.

 Використання 
Вставте шорткод у будь-яке місце запису або сторінки:

[cpi_posts count="5" sort="date"]

 Параметри  
- `title` — заголовок блоку (за замовчуванням: "Latest Posts")  
- `count` — кількість постів для відображення (за замовчуванням: 5)  
- `sort` — поле сортування: `date`, `title`, `modified`, `rand`  
- `ids` — список ID постів через кому (наприклад: `12,15,18`)
